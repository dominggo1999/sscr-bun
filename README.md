# sscr

Show scripts from package.json

![Demo](/ss.gif "Demo")

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

Install globally

```bash
npm install -g .
```

This project was created using `bun init` in bun v0.3.0. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
